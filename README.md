# chatterbox

A modern chatroom application built with Next.js, Supabase, and Shadcn UI.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Shadcn UI
- **Database/Auth:** Supabase
- **Payments:** Stripe
- **Email:** Resend
- **State Management:** Zustand
- **Deployment:** Cloudflare Pages
- **Icons:** Lucide React
- **UI Components:** Radix UI
